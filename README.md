# WinForge

WinForge is a Windows 10/11 setup, optimization and maintenance toolkit built with PowerShell 5.1.

## Features

- Application installer (winget)
- Windows tweak system (safe / advanced / risky)
- Maintenance tools (SFC, DISM, cleanup)
- Backup & rollback system
- PowerShell 5.1 compatible

## Requirements

- Windows 10 / 11
- PowerShell 5.1
- Administrator privileges
- Internet connection (for winget)

## Usage

```powershell
powershell -ExecutionPolicy Bypass -File .\WinForge.ps1
